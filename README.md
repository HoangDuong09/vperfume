# VPerfume

VPerfume là một website thương mại điện tử bán nước hoa cao cấp được phát triển bằng ReactJS. Dự án tập trung vào trải nghiệm người dùng thân thiện, giao diện hiện đại và tối ưu trên nhiều thiết bị, cho phép khách hàng duyệt sản phẩm, xem chi tiết, thêm vào giỏ hàng và dùng tài khoản đăng nhập đơn giản.

## Demo

Live Demo: vperfume.vercel.app

## Công nghệ sử dụng

- ReactJS
- JavaScript (ES6+)
- HTML5
- CSS3
- React Router DOM
- Context API
- Local Storage

## Tính năng nổi bật

- Hiển thị danh sách nước hoa
- Xem chi tiết sản phẩm
- Tìm kiếm sản phẩm
- Lọc sản phẩm theo danh mục
- Sắp xếp sản phẩm
- Slider banner tự động
- Giỏ hàng
- Đăng ký tài khoản
- Đăng nhập
- Đăng xuất
- Lưu dữ liệu bằng Local Storage
- Responsive trên Mobile, Tablet và Desktop
