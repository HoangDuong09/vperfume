# VPerfume

VPerfume là một website thương mại điện tử bán nước hoa cao cấp được phát triển bằng ReactJS. Dự án tập trung vào trải nghiệm người dùng thân thiện, giao diện hiện đại và tối ưu trên nhiều thiết bị, cho phép khách hàng duyệt sản phẩm, xem chi tiết, thêm vào giỏ hàng và dùng tài khoản đăng nhập đơn giản.
